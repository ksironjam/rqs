Introduction

jRQS is a free open-source program that allows you to search for the requirements parsed from Excel spreadsheets.

This software allows you to make the preliminary search for the requirements only. You should always keep in mind that search results might be outdated.


How to search

It is possible to search for the requirements by FR ID, TMS Task, or text. You may specify a single FR ID, TMS Task or keyword to search as well as several values separated by a comma. There are a few examples indicated below to help get you involved:

Example Description
FR1 Search for a concrete requirement by FR ID
FR1,FR2,FR3 Search for a bunch of requirements by FR ID
PCC-12345 Search for requirements by TMS task number
PCC-11111,PCC-22222 Search for requirements by several TMS task numbers
keyword Search for requirements by keyword
keyword1,keyword2 Search for requirements by several keywords. AND logic is being applied to the search, so all the requirements containing both keywords will be found.

All the search is case-insensitive.

Search results are limited to 100. If your search returned 100 or more results, then you will be notified by an appropriate message.


Column descriptions

Column name Description
FR ID Requirement's unique identifier per project
TMS Task TMS task number associated with the requirement
Object Number Interaction level of the requirement
Text Requirement's text itself
CCP Level of the failure, where:
 5 is a critical, leading to serious permanent injury or death,
 4 is critical, could cause serious delay or misinformation,
 3 is a failure that could cause to extended delay in patient care,
 2 is a failure that could cause to delay in treatment of patient,
 1 is a minor nuisance to users
Created / Modified The dates when requirement was created and modified last time
Is changed? The indicator that shows are created and last modified dates equal or not
Status Requirement's creation/review status
Boundary
Source The source file name where from the requirement was imported into the jRQS database
